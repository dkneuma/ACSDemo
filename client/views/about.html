ACS Technologies

Front End Challenge

This site is a single-page application that serves as a front-end to the ACS Technologies challenge API

The Home page displays a list of contacts.

Users can enter search criteria in the search field. This will filter the list

Users can click on one of the field headings to sort the contacts by that field.

Clicking on the name of the contact will take the user to a page where they can view and edit all of the contact info.

Note: When editing an existing contact, or adding a new contact, the following validation is enforced:

  • First Name is required
  • Email must be blank or have a valid format
  • URL must be blank or have a valid format
